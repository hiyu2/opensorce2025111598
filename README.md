## 기능설명

총 5개의 기능이 있다.
단위변환기, 간단 계산기, 점수 평균 계산, 문자열 처리, 랜덤 비밀번호 생성기.

단위변환기는 cm단위를 입력하면 m단위로 변경한다.
간단 계산기는 수식 전체를 입력하면 그 수식을 계산해 준다. 단, 계산은 사칙연산만 가능하다.
점수 평균 계산은 한줄에 점수를 입력하고, 점수가 모두 입력된 후, -1을 입력하면 평균이 나온다.
문자열 처리는 문자열을 입력받고, 소문자들이 대문자로 변환된다. 이후, 대문자를 입력하면 대문자들이 소문자로 변환된다.
랜덤 비밀번호 생성기는 랜덤으로 12자리의 대소문자와 특수문자가 입력된 비밀번호가 나온다.
